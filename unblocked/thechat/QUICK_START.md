# TheChat Quick Start Guide

## 🚀 Getting Started (5 Minutes)

### Step 1: Clear Your Browser Cache
**Important:** If you had the old version open, you MUST do this:
- Press `Ctrl + Shift + R` (Windows/Linux)
- Or `Cmd + Shift + R` (Mac)
- This forces a fresh load of the new code

### Step 2: Access TheChat
1. Make sure you're logged in to your account
2. Navigate to: `/unblocked/thechat/`
3. You should see the Direct Messages view

### Step 3: Set Up Your Profile
1. Click your avatar in the bottom left corner
2. Click "Edit Profile"
3. Set your:
   - Username
   - Bio (optional)
   - Status (Online, Idle, DND, Invisible)
   - Profile Picture URL (optional)
4. Click "Save"

## 💬 Using Direct Messages

### Start a DM
1. You need to be friends first
2. Click on a user's profile (from server member list)
3. Click "Add Friend"
4. Then click "Send Message"
5. DM opens automatically

### View Your DMs
1. Click the 💬 button at the top of the left sidebar
2. You'll see:
   - Friends button
   - List of active DMs
3. Click any DM to open it

## 🖥️ Using Servers

### Create Your First Server
1. Click the **+** button in the left sidebar
2. Enter a server name (e.g., "My Server")
3. Enter an icon (emoji or 2 letters, e.g., "🎮" or "MS")
4. Click "Create"
5. **Copy the Server ID** - you'll need this to share
6. Server appears immediately in left sidebar

### Join a Server
1. Get a Server ID from a friend
2. Click the compass button in left sidebar
3. Paste the Server ID
4. Click "Join Server"
5. Server appears in left sidebar

### Switch Between Servers
1. Click any server icon in the left sidebar
2. Server name appears at top
3. Channels list appears in middle
4. Members list appears on right

### Chat in a Server
1. Click a server in left sidebar
2. Click a channel (e.g., #general)
3. Type your message at the bottom
4. Press Enter to send

## 👤 User Interactions

### View Someone's Profile
- Click their username in chat
- Click their avatar anywhere
- Click them in the member list

### Add a Friend
1. Open their profile
2. Click "Add Friend"
3. They'll receive a friend request

### Block a User
1. Open their profile
2. Click "Block User"
3. Confirms and blocks them

### Start a DM with Someone
1. Open their profile
2. Click "Send Message"
3. DM opens automatically

## ⚙️ Settings & Status

### Change Your Status
1. Click your avatar (bottom left)
2. Click "Edit Profile"
3. Select status:
   - 🟢 Online
   - 🟡 Idle
   - 🔴 Do Not Disturb
   - ⚫ Invisible
4. Click "Save"

### Update Your Bio
1. Edit profile
2. Type in bio field (max 190 characters)
3. Save

### Change Your Avatar
1. Upload an image to any image host
2. Copy the direct image URL
3. Edit profile
4. Paste URL in "Profile Picture URL"
5. Save

## 💡 Tips & Tricks

### Keyboard Shortcuts
- `Enter` - Send message
- `Shift + Enter` - New line in message
- `Escape` - Close modals

### Message Actions
- **Edit** - Hover over your message → Click ⋯ → Edit
- **Delete** - Hover over your message → Click ⋯ → Delete
- **Send GIF** - Click image icon → Search → Select

### Server Management
- **Get Server ID** - Click server name → Menu button → Shows ID
- **Share Server** - Give friends the Server ID
- **Leave Server** - (Coming soon)

### Profile Tips
- Add a custom avatar to stand out
- Set a fun bio to tell others about yourself
- Change status when AFK or busy
- Click any username to see their profile

4. **Type "Hello!"** and press Enter
5. **Invite friends** by sharing Server ID

## 🔧 If Something's Wrong

### Nothing Showing Up?
- Hard refresh: `Ctrl + Shift + R`
- Check you're logged in
- Check browser console (F12) for errors

### Can't Send Messages?
- Make sure you're in a channel or DM
- Check you have internet
- Verify Firebase rules are deployed

### Server Not Loading?
- Hard refresh
- Try rejoining the server
- Check Server ID is correct
