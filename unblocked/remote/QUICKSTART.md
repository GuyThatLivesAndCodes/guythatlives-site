# 🚀 Remote Play - Quick Start

Get Hyperbeam remote gaming running in 5 minutes - **100% FREE** with Cloudflare Workers!

## 🎯 Choose Your Setup Method

### 🔥 **Option 1: Cloudflare Workers (RECOMMENDED)** ⭐

**Best for:** Everyone! No credit card, 100% free, super easy.

✅ **Pros:**
- Completely free (100k requests/day)
- No credit card required
- 5-minute setup
- API key stays secure

**Follow:** `CLOUDFLARE-SETUP.md` (in this folder)

**Quick Steps:**
1. Get Hyperbeam API key (2 min)
2. Create Cloudflare Worker (3 min)
3. Update `fortnite.html` with worker URL (30 sec)
4. Deploy & test!

---

### 💳 Option 2: Firebase Functions (Requires Blaze Plan)

**Best for:** If you already have Firebase Blaze plan active.

⚠️ **Requires:**
- Firebase Blaze (pay-as-you-go) plan
- Credit card on file
- More complex setup

**Follow:** `SETUP.md` (in this folder)

---

## 🏆 We Recommend: Cloudflare Workers!

Since Firebase Functions require the Blaze plan (and a credit card), **Cloudflare Workers** is the better choice for most users.

### Why Cloudflare?

| Feature | Cloudflare | Firebase |
|---------|-----------|----------|
| Free Tier | ✅ 100k/day | ❌ Requires Blaze |
| Credit Card | ❌ Not needed | ✅ Required |
| Setup Time | ⚡ 5 min | 🕐 15 min |
| Security | 🔒 Secure | 🔒 Secure |
