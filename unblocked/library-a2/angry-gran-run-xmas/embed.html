Angry Gran Run Xmas is a very exciting and festive, endless runner style game that places you in the running shoes of granny as you run through the Christmas inspired town, collecting coins and dodging dangerous obstacles as you go! Jump over candy canes and slide under Christmas trucks in order to make it through the town. Use the coins that you collect to unlock new outfits and characters to run with. Can you get the highest score or will your run be cut short?
