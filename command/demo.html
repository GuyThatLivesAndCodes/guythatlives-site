üìñ Quick Start Commands

help
Display all available commands with usage examples
open <url>
Open a URL in a new draggable window
Example: open https://example.com
inspect <window-id>
Toggle console panel for a specific window
Example: inspect win-1
list
Show all currently open windows with their IDs
focus <window-id>
Bring a specific window to the front
Example: focus win-2
close <window-id>
Close a specific window
Example: close win-1
clear
Clear the terminal output history

üí° Tips & Tricks

Keyboard Shortcuts:
  • ‚Üë/‚Üì Arrow Keys: Navigate command history
  • Tab: Auto-complete command names
  • Enter: Execute current command
Window Interactions:
  • Drag: Click and hold the window header to move
  • Resize: Drag from the bottom-right corner
  • Focus: Click anywhere on a window to bring it forward
  • Yellow Button: Toggle console inspector
  • Red Button: Close the window

‚ö†Ô∏è Known Limitations

CORS Restrictions: Many websites cannot be loaded in iframes due to security headers (X-Frame-Options). This is a browser security feature, not a limitation of the terminal. Sites like Google, Facebook, and banks typically block iframe embedding.
Console Access: The console inspector cannot capture logs from cross-origin iframes due to CORS policies. It will show basic page load events and any errors that occur during loading.
